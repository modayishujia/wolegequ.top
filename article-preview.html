全流程复盘

下面把整个流程走一遍，给同样想搞站的朋友参考。

1写代码，建网站

这次没有用 WordPress，直接上的 Astro + Tailwind CSS。

为什么？

  • 静态站速度快，Google 喜欢
  • 不需要服务器，省 hosting 钱
  • 全部代码在本地，Git 管理，心里踏实

主站更简单，纯 HTML + CSS + JS，连构建工具都没用。为什么？因为——垃圾站就要有垃圾站的样子，越简单越好维护。

16个赛道的起号指南内容，每个赛道都是三四千字的深度长文，包含：

  • 赛道深度分析
  • 账号搭建指南
  • 内容策略和选题方向
  • 涨粉技巧
  • 变现路径
  • 30天冷启动计划
  • 常见误区和避坑指南

内容才是王道，SEO 的尽头是好内容。（虽然我自己也不太信这句话）

2申请 Google AdSense

这一步是玄学中的玄学。

第一次申请，被拒了。理由是"内容不足"。

于是我疯狂加内容：隐私政策、关于我们、联系方式、使用条款、博客文章……能加的全加上。中英文双版本，五种语言切换，47个页面打底。

第二次申请，过了。

AdSense 过审经验总结：

1. 页面数量：至少 15-20 个有实质内容的页面

2. 必要页面：隐私政策、关于、联系、使用条款，一个都不能少

3. 原创内容：别抄，Google 不傻

4. 导航清晰：让用户（和 Google 爬虫）能轻松找到所有页面

5. 多语言：有条件就上，显得专业

3域名解析迁移

这是整个流程中最刺激的一步。

我的域名 wolegequ.top 原来在阿里云注册，DNS 也在阿里云。为了用 Cloudflare Pages 部署，需要把域名的 nameserver 迁移到 Cloudflare。

操作步骤：

① 在 Cloudflare 后台添加域名，它会给你两个 nameserver 地址：

ricardo.ns.cloudflare.com
sandy.ns.cloudflare.com

② 去阿里云的域名注册控制台（注意不是 DNS 解析控制台！这是两个不同的地方！）

  • 路径：域名注册 → 域名详情 → DNS 管理 → DNS修改
  • 点击"修改DNS服务器"
  • 输入 Cloudflare 给的两个 nameserver
  • 需要短信验证（阿里云绑定的手机号）

③ 等待 DNS 传播，一般 24-48 小时

踩坑记录：阿里云的"解析设置"和"域名注册"是两个完全不同的控制台。nameserver 的修改必须在"域名注册"里操作，不是在"解析设置"里加 CNAME 记录就完事了。我在这一步卡了好久。

4发布到 GitHub

代码版本管理用 GitHub，这个没什么好说的。

git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/modayishujia/wolegequ.top.git
git push -u origin main

子站也各有各的仓库：

  • modayishujia/wolegequ.top
  • modayishujia/qihao.wolegequ.top
  • modayishujia/download.wolegequ.top

GitHub 不仅是代码仓库，更是 Cloudflare Pages 自动部署的触发器——每次 push 到 main 分支，Cloudflare 就会自动拉取代码、构建、部署。

5Cloudflare Pages 部署

这一步是最爽的，因为——免费。

Cloudflare Pages 的免费套餐：

  • 无限站点
  • 无限请求
  • 无限带宽
  • 每月 500 次构建
  • 100 个自定义域名

对比一下 Vercel 和 Netlify 的免费套餐，Cloudflare 简直是慈善家。

部署步骤：

  • 登录 Cloudflare Dashboard
  • Pages → Create a project → Connect to Git
  • 选择 GitHub 仓库
  • 配置构建设置：
    • 主站（纯 HTML）：构建命令留空，输出目录 /
    • 起号指南（Astro）：构建命令 npm run build，输出目录 dist
  • 点部署，等一分钟，完成

部署完会得到一个 xxx.pages.dev 的默认域名，然后在 Custom Domains 里绑定自己的域名。

因为域名已经在 Cloudflare 了，添加自定义域名的时候会自动创建 CNAME 记录，秒生效，不用手动配置 DNS。
